<ul dir="auto">
<li>Open the extension and click one of the <code>Choose File</code> buttons to select device files</li>
<li>Select the type of device you're using in the top <code>Systems</code> section</li>
<li>The files are saved in the extension's <code>chrome.storage.sync</code> storage and will be synchronized across any browsers into which the user is signed in with their Google account.</li>
